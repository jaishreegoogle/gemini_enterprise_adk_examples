# Gemini Enterprise ADK Examples

This repository contains examples and utilities for the Gemini Enterprise ADK.

## Project Structure

Here's a brief overview of the key files and their purposes in this repository:

*   **`.github/`**: Contains GitHub-specific configurations, possibly for workflows or issue templates.
*   **`__init__.py`**: This file initializes the Python package by importing `root_agent` from the `agent` module.
*   **`agent.py`**: This file defines the `GitHubExpert` AI agent, which specializes in GitHub data fetching and documentation workflows, and configures its tools and instructions.
*   **`localtest.md`**: A markdown file, possibly for local testing documentation or notes.
*   **`test server.py`**: This file contains the implementation of the `FastMCP` server and defines several GitHub-related tools for reading file content, getting project dependencies, retrieving repository information, listing repository contents, analyzing READMEs, finding README targets, creating branches, committing files, and creating pull requests.
